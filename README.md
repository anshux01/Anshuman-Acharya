# 🚀 Anshuman Acharya - Portfolio Website

A modern, interactive portfolio website showcasing my journey as a Full-Stack Developer and Data Science Aspirant. Built with clean HTML, CSS, and JavaScript featuring smooth animations, responsive design, and an elegant cream & blue color palette.

![Portfolio Preview](img/portfolio-preview.png)

## 🌟 Features

### ✨ Interactive Design
- **Floating Particles Animation** - Dynamic background particles for visual appeal
- **Scroll Progress Indicator** - Real-time scroll tracking
- **Smooth Hover Effects** - Interactive cards and buttons with transforms
- **Loading Animation** - Professional loading screen on page load
- **Animated Avatar Ring** - Rotating gradient border around profile picture

### 🎨 Modern UI/UX
- **Color Hunt Palette** - Cream & Blues theme (`#FFFBDE`, `#91C8E4`, `#749BC2`, `#4682A9`)
- **Card-Based Layout** - Clean, organized content presentation
- **Responsive Design** - Optimized for desktop, tablet, and mobile
- **Typography** - Beautiful Poppins font from Google Fonts
- **Glassmorphism Effects** - Subtle blur and transparency effects

### 📱 Responsive Features
- Mobile-first approach
- Adaptive grid layouts
- Touch-friendly interactive elements
- Optimized image sizes for different screen sizes

## 🛠️ Technologies Used

- **HTML5** - Semantic markup structure
- **CSS3** - Advanced styling with custom properties, animations, and grid
- **Vanilla JavaScript** - Interactive functionality and DOM manipulation
- **Boxicons** - Icon library for consistent iconography
- **Google Fonts** - Poppins font family

## 🔧 Customization Guide

### Changing Colors
Update the CSS custom properties in the `:root` selector:
```css
:root {
  --cream: #FFFBDE;
  --light-blue: #91C8E4;
  --medium-blue: #749BC2;
  --dark-blue: #4682A9;
}
```

### Adding New Sections
1. Create a new card in the grid
2. Add appropriate icon from Boxicons
3. Style with existing CSS classes
4. Add any custom animations if needed

### Modifying Animations
Adjust animation parameters in CSS:
```css
/* Example: Change particle animation speed */
@keyframes floatDot {
  /* Modify duration in animation property */
}
```
